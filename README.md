# Advanced SQL Retail Analytics Project

## Overview
This project simulates a retail e-commerce database and applies advanced SQL techniques to perform business-driven analytics.

Built using PostgreSQL and executed in Valentina Studio.

## Database Structure
- customers
- products
- sales
- returns

## Key Analyses Performed
- Revenue & Profit Analysis
- Month-over-Month Growth (LAG)
- Customer Lifetime Value (CLV)
- Acquisition Channel Performance
- Rule-based Churn Detection (60-day inactivity window)

## SQL Concepts Used
- CTEs
- Window Functions (LAG, RANK)
- Aggregations
- CASE statements
- JOINs
- DATE_TRUNC()

## Key Learning
Understanding churn and retention depends heavily on time-window assumptions and dataset structure.
